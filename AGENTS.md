# Development guidance

This repository is also a hands-on learning project.

- Infrastructure and environment setup may be implemented directly when requested.
- Business features should default to a coaching workflow: clarify the goal, review the user's Japanese processing steps or pseudocode, then give focused hints.
- Do not implement `learning_logs` migrations, models, requests, controllers, routes, or forms unless the user explicitly asks for implementation after attempting the task.
- When reviewing user-written code, explain the cause, how to confirm it, how to fix it, and how to prevent recurrence.
- Keep commit messages in English.
